# jump-game
使用php 玩微信跳一跳(-. .-)

- 目前只测试自己的手机，华为 720*1280,中心点跳中概率大概95%

alphaed: 去除背景，杂质图片存储目录
image: 各点图片存储目录

## 思路

 - 根据棋子颜色获取棋子顶点，中心点，棋子上亮色点(用来修正棋子位置)
 - 将背景变黑
 - 棋盘位置就是棋盘顶点x坐标，棋盘左边极点y坐标

## 依赖

 - php-cli (>=7.0)
 - php-gd
 - adb 调试工具
 - android 手机
